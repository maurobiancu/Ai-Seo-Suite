Genera automaticamente titoli, descrizioni e meta tag SEO con l’aiuto dell’Intelligenza Artificiale. Compatibile con Yoast SEO.

== Descrizione (IT) ==
**AI SEO Suite** è il plugin WordPress che ti aiuta a ottimizzare i contenuti del tuo sito grazie all’Intelligenza Artificiale.  
Con un clic puoi generare:
- Titoli ottimizzati
- Meta description efficaci
- Parole chiave rilevanti

Integrazione con **Yoast SEO**: i contenuti generati possono essere applicati direttamente ai campi del plugin Yoast, risparmiando tempo e migliorando la visibilità nei motori di ricerca.

== Caratteristiche principali (IT) ==
- Generazione automatica di meta title, description e keyword
- Pulsante per applicare i contenuti a Yoast SEO
- Risparmio di tempo e maggiore efficienza
- Interfaccia semplice e intuitiva

== Installazione (IT) ==
1. Carica la cartella del plugin nella directory `/wp-content/plugins/`  
2. Attiva il plugin dal menu "Plugin" in WordPress  
3. Vai alla pagina di impostazioni **AI SEO Suite** e inizia a generare i tuoi contenuti SEO  

== Frequently Asked Questions (IT) ==
= Funziona solo con Yoast SEO? =  
No, i contenuti generati possono essere usati anche senza Yoast. L’integrazione semplifica solo l’applicazione automatica.

= Posso modificare i testi generati? =  
Sì, i contenuti sono sempre modificabili prima di essere applicati.
